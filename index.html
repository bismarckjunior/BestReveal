Best Reveal

Apresentação dinânica de slides usando HTML

https://bismarckjunior.github.io/BestReveal

O que é?

Best Reveal é uma forma dinânica de apresentação de slides utilizando HTML baseado no framework Reveal.js.

Veja um exemplo usando Reveal.js!

Por que usar o BEST REVEAL?

Best Reveal possui todas as funcionalidades do Reveal.js com uma enorme coletânea de plugins já instalados. É baixar e usar.
